# 🏟️ 점심아레나 (Noon Arena) - 게임 기획서

## 컨셉
점심 먹고 커피내기용 모바일 웹 배틀로얄. 둥근 아레나에서 밀치고, 돌진하고, 회피해서 상대를 밖으로 떨어뜨려라!

## 핵심 게임 메카닉

### 아레나
- 원형 플로팅 아레나 (떠있는 섬 느낌)
- 시간이 지나면 아레나가 점점 줄어듦 → 긴장감 UP
- 아레나 밖 = 낙사 = 탈락

### 캐릭터
- 동글동글한 귀여운 아바타 (이모지 기반)
- 각 플레이어 랜덤 이모지 + 닉네임
- 물리: 원형 충돌체, 질량 기반 밀림

### 조작 (모바일 터치 최적화)
- **이동**: 가상 조이스틱 (왼쪽 엄지)
- **돌진(Dash)**: 대시 버튼 (오른쪽 엄지) - 전방 돌진, 쿨다운 3초
- **밀치기(Push)**: 충돌 시 자동 + 대시 충돌 시 강한 넉백
- **회피(Dodge)**: 스와이프 또는 더블탭 - 짧은 무적 + 이동, 쿨다운 5초

### 게임 플로우
1. **로비**: 방 코드로 입장 (4자리), 최대 10명
2. **대기실**: 접속자 목록, 방장이 시작
3. **라운드**: 60초 제한, 아레나 30초부터 수축
4. **결과**: 최후의 1인 승리, 순위표 표시
5. **재경기**: 바로 다시 시작 가능

### 중독 요소
- 빠른 라운드 (60초)
- 아레나 수축으로 숨을 곳 없음
- 대시 타이밍의 스킬 요소
- 이모지 아바타의 재미
- 킬 로그 + 효과음
- 연승 카운터

## 기술 스택
- **서버**: Node.js + Socket.io (실시간 동기화)
- **클라이언트**: HTML5 Canvas + 바닐라 JS (단일 HTML)
- **물리**: 자체 2D 원형 충돌 (간단한 물리엔진)
